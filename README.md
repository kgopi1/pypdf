Merge multiple PDF files into single file using Python Script. 

Steps:
1. Install python package requriements.
   ```
    pip install -r requirements.txt 
   ```
2. Add pdf files in to pdf directory
3. update file names in the pdf/merge.py script.
4. Run the python script.
   ```
   python merge.py
   ```
5. Combined pdf files saved into result.pdf.
